Arrays

Linear Arrays

A linear array is a list of finite numbers of elements stored in the memory. In a linear array, we can store only homogeneous data elements. Elements of the array form a sequence or linear list that can have the same type of data. Each element of the array is referred by an index set.

Operations on Arrays

Following are the operations on Linear Array:
1.Traversing.
2.Searching.
3.Insertion.
4.Deletion.
5.Sorting.
6.Merging.

1.Operations on Array: Traversal

Traversal means visiting each element of array exactly once. Used for processing of array like printing the contents or counting the no. of elements in array.

Algorithm
[initialize counter] set k = LB
repeat steps 3 and 4 while k<= UB
[visit the element] apply process to a[k]
[increase counter] set k = k + 1
[end of step 2 loop]
exit

Alternative Algorithm (Repeat Until Loop)
repeat for k = LB to UB
Apply process to A[k]
[end of loop]
exit
