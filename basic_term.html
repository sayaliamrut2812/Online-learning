BASIC TERMINOLOGIES



  • Data:
          Data is simply a value or a set of values of different types like integer, string, char , etc.


  • Structure:
          Structure is a way of organizing information / data, so that it becomes easier to use.


  • Data Structure:
          Data structures are structures programmed to store ordered data, so that various operations can be performed easily.

    •       -It should be designed in such a way that it reduces the complexity and increases the efficiency.


    Why Data Structure?

  •    To solve the complex requirements efficiently we need data structures.

  •    It provides fastest solution of human requirements.

  •    It provides efficient solution of complex problems with respect to

            -Space

            -Time
